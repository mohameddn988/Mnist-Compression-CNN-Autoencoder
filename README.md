MNIST Compression and Reconstruction using Convolutional Autoencoders
This project demonstrates image compression using deep learning, specifically convolutional autoencoders, on the MNIST handwritten digits dataset. It shows how images can be encoded into a smaller latent representation and then decoded back, visualizing the reconstruction process and assessing the quality of compression.
